# Restaurante Martilota

Web del Restaurante Martilota (Alcalá de Henares).

Versión online: https://jenn-jt.github.io/martilota/

## Páginas

- Inicio: espacios del restaurante, carta destacada y reserva.
- Carta: platos por categorías con alérgenos según Reglamento UE 1169/2011.
- Dónde estamos: mapa, contacto y entorno del casco histórico.

## Identidad visual

- Tipografías: Cormorant Infant y Libre Franklin
- Paleta: terracota `#CB6037`, crema `#F5F7F7`, negro `#1A1A1A`
